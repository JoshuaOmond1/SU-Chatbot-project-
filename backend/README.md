# Unified FastAPI service

## Run locally

```powershell
python -m venv .venv
.venv\Scripts\pip install -e ".[dev]"
Copy-Item .env.example .env
.venv\Scripts\uvicorn app.main:app --reload
```

The no-key development mode returns a safe fallback response. Production uses
passwordless IAM authentication to the Firebase SQL Connect Cloud SQL database.
Set `USE_DATABASE=true` with the Cloud SQL values in `.env.example`; query embeddings
are generated by Vertex AI using the same model and dimension as SQL Connect.

Approved text exports can be loaded as inactive drafts with `python -m app.rag.ingest`.
The Finance, Registrar, or other owning office must review and activate each source
and its chunks in Firebase before retrieval will return them.

Identity flow: the existing website/app obtains a one-use CAS ticket or AD/OIDC
access token, calls `POST /v1/auth/exchange`, and receives a 15-minute assistant
JWT. The API validates the institutional credential server-to-server; it does not
accept a student number or email as proof of identity.
